# To Do List Prototype

Proyek sederhana untuk membuat aplikasi to-do list menggunakan HTML, CSS, dan JavaScript. 
Tujuan utama: belajar manipulasi DOM, event handling, dan pengelolaan elemen dinamis.

## Fitur
- Menambahkan task baru melalui input.
- Menghapus task tertentu dengan tombol hapus.
- Menampilkan pesan "tugas tidak ditemukan" saat daftar kosong.

## Struktur Project
- `index.html` → struktur halaman.
- `style.css` → styling tampilan.
- `main.js` → logika tambah/hapus task.

## Rencana Pengembangan
- Level 8: fitur "mark as done" (centang tugas selesai).
- Level 9: animasi dan styling interaktif.
- Level 10: simpan data di localStorage agar daftar tetap ada setelah refresh.

## Lisensi
Proyek ini dibuat untuk tujuan belajar. Bebas digunakan dan dimodifikasi.
